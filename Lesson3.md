# Flutter ウィジェットのレイアウト

## 1. テキストスタイル

| ウィジェット | 説明 |
|------------|------|
| `fontSize` | フォントサイズ |
| `fontWeight` | フォントの太さ |
| `fontFamily` | フォントファミリー |
| `color` | テキスト色 |

## 2. Conteinerクラス

| ウィジェット | 説明 |
|------------|------|
| `Color` | 色の指定 |
| `Alignment` | 配置場所の指定 |
| `Sized` | 表示サイズを最大化 |
| `Padding` | 余白幅の設定 |

## 3. Alignmentについて

| ウィジェット | 説明 |
|------------|------|
| `topLeft` | 左上 |
| `topCenter` | 中央上 |
| `topRight` | 右上 |
| `centerLeft` | 左中央 |
| `center` | 中央 |
| `centerRight` | 右中央 |
| `bottomLeft` | 左下 |
| `bottomCenter` | 中央下 |
| `bottomRight` | 右下 |
